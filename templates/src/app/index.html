Does this page look dull and un-Storebrandy?

Make sure you have a compiled css of the storebrand styles in your /bower_components/stb-webmanual/css folder. 

                      $> bower install
                      $> cd bower_components/stb-webmanual/less
                      $> lessc stb-main.less > ../css/stb-main.css
